RUS:
    Внимание!!!
        У вас возможно не установлена библиотека pygame
        Как это исправить?
            открываем консоль Win + R -> cmd -> Enter (на линукс: Ctrl + Alt + T), там прописываем pip install pygame (или pip3 install pygame)
        Программу запускать только через "main.py"
        Консоль, открывающуюся при запуске не закрывать!!!
    Подробности о версии игры читайте в version.txt
    
ENG:
    Attention!!!
    You may not have the pygame library and keyboard library installed
        How can I fix this?
            open the console Win + R -> cmd -> Enter (on Linux: Ctrl + Alt + T), there we register pip install pygame (or pip3 install pygame)
        Run the program only through "main.py"
        Do not close the console that opens at startup!!!
    Read more about the version of the game in version.txt
